\doxysubsection{Detailed Description}
Represents an ordered set of points in 2D, forming a polyline (open or closed). 

The {\ttfamily \doxylink{classhmap_1_1Path}{Path}} class extends the {\ttfamily \doxylink{classhmap_1_1Cloud}{Cloud}} class to represent a sequence of points in 2D space that are connected in a specific order, forming a polyline. The polyline can be either open or closed, depending on whether the first and last points are connected. This class provides methods for various geometric operations, including path length calculation, point insertion, and more.

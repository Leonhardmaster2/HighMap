\doxysubsection{Detailed Description}
Storage manager for \doxylink{classhmap_1_1VirtualTexture}{Virtual\+Texture}. 

A \doxylink{classhmap_1_1VirtualTexture}{Virtual\+Texture} is composed of multiple Virtual\+Arrays (channels). This storage wrapper ensures that each channel has a proper \doxylink{classhmap_1_1TileStorage}{Tile\+Storage} (RAM, Disk\+LRU, Sequential) and can manage shared directories, cleanup, etc.

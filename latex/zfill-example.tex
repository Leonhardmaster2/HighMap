\doxysection{zfill}
\hypertarget{zfill-example}{}\label{zfill-example}Pads the beginning of a string with zeros until it reaches a specified length.

Pads the beginning of a string with zeros until it reaches a specified length.\+This function prepends \textquotesingle{}0\textquotesingle{} characters to the input string {\ttfamily str} so that the resulting string has at least {\ttfamily n\+\_\+zero} characters. If {\ttfamily str} is already equal to or longer than {\ttfamily n\+\_\+zero}, it is returned unchanged.


\begin{DoxyParams}{Parameters}
{\em str} & The input string to pad. \\
\hline
{\em n\+\_\+zero} & The minimum total length of the resulting string after padding. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
A new string padded with leading zeros to reach the specified length.
\end{DoxyReturn}
("{}42"{}, 5); // returns "{}00042"{} zfill("{}12345"{}, 5); // returns "{}12345"{}

\doxysubsection{Detailed Description}
Path class for manipulating and analyzing paths in 2D space. 

\begin{DoxyAuthor}{Author}
Otto Link (\href{mailto:otto.link.bv@gmail.com}{\texttt{ otto.\+link.\+bv@gmail.\+com}}) 
\end{DoxyAuthor}
This file defines the {\ttfamily Path} class, which extends the {\ttfamily Cloud} class to provide functionalities specific to paths. The {\ttfamily Path} class includes methods for creating, manipulating, and analyzing paths, including smoothing the path with various curves, resampling, and performing operations like meandering and fractalizing.
